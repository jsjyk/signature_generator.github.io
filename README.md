# ✍️ Online Signature Pad
> 마우스나 터치로 간편하게 서명을 생성하고 PNG 파일로 저장하세요.

**[라이브 데모 바로가기](https://jsjyk.github.io/signature_generator.github.io/)**

### ✨ 주요 기능
* **부드러운 드로잉:** HTML5 Canvas를 이용한 매끄러운 서명 경험
* **원클릭 다운로드:** 작성한 서명을 투명 배경의 PNG 파일로 즉시 저장
* **캔버스 초기화:** 서명이 마음에 들지 않으면 언제든 다시 시작
* **반응형 디자인:** PC와 모바일 기기 완벽 지원
